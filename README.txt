voici quelques lignes explicatives quant à la structure du site:

Structure de répertoires
************************

htdocs/ (publié)		
	css/ (contient les feuilles de style)
	js/ (contient le code javascript éventuel)
	img/ (contient les images et autres graphiques)
	index.php (fichier principal de l'application)

includes/ (non plublié)
	class/ (contient les définitions de classes PHP, p.ex. Tamagoshi.php)
	templates/ (contient les templates HTML)
	templates_c/ (contient les templates HTML compilés par Smarty)

config/ (non plublié, contient les fichiers de configuration)
	connexion.php (fichier contenant des données sensibles)
